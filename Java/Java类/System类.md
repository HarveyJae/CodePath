# System类的概念：
- `System`类是系统类，提供了很多**基础方法**：
   - 标准输入、标准输出和错误输出流。
   - 访问外部定义的属性和环境变量。
   - 加载文件和库的手段。
   - 用于快速复制数组部分的实用方法。
- System类的标签：`public final class System extends Object`
   - `System`类是`final`类，不允许继承。
   - `System`类不允许实例化。
- `System`类中的方法/属性(不包括继承于`Object`的方法/属性)均为**静态方法/属性**。
